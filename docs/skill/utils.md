# 可能会用到的一些工具

## verdaccio

用来搭建私库。

提供了一个基本的组件页面，但是写接口文档过于麻烦，可以使用 dumi

## father

组件/库 构建打包工具

## dumi

- 组件/库 项目脚手架。生成的项目模板包括组件以及组件文档(可在线预览，加入了 sandbox)。

- 使用 father 构建、打包。

- 可打包 npm 包和组件/库文档网站包。
